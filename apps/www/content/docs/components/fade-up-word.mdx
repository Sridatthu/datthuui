## Props

| Prop         | Type                           | Default | Description                                    |
| ------------ | ------------------------------ | ------- | ---------------------------------------------- |
| `text?`      | `string`                       | `""`    | The text content to display                    |
| `className?` | `string`                       | `""`    | Additional CSS classes to apply to the element |
| `as?`        | `"h1" \| "h2" \| "h3" \| "h4"` | `"h2"`  | add you h tag                                  |
